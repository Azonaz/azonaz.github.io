## How Fibonacci Merges Work

Fibonacci Hex Merge is a Fibonacci merge puzzle built around a familiar number sequence.

Place numbers on the hex board. When neighboring numbers follow the Fibonacci sequence, they merge into the next number:

- `1 + 2 = 3`
- `2 + 3 = 5`
- `3 + 5 = 8`
- `5 + 8 = 13`

Each merge gives you points and opens space on the board. Longer chains can turn one careful placement into a stronger score.

## Why The Hex Board Matters

The hex board changes how a number puzzle feels.

Each cell has more neighbors than a square grid, so every placement can create several possible paths. A good move is not only about the number you place now. It is about the chain you leave open for the next few turns.

That makes Fibonacci Hex Merge a compact hex merge puzzle: easy to start, but better when you slow down and plan.

## FAQ

### Is Fibonacci Hex Merge an offline number puzzle?

Yes. Fibonacci Hex Merge is an offline number puzzle for iPhone. You can play without Wi-Fi or a mobile connection.

### Does Fibonacci Hex Merge have ads?

No. Fibonacci Hex Merge has no ads.

### Does the game track me?

No. Fibonacci Hex Merge does not collect personally identifiable information.

### How do Fibonacci merges work?

Numbers merge when they follow the Fibonacci sequence. For example, `1 + 2` makes `3`, `2 + 3` makes `5`, and `3 + 5` makes `8`.

### Is Fibonacci Hex Merge like Sudoku?

It is not Sudoku and does not use Sudoku rules. It may appeal to players who enjoy quiet number logic, planning ahead, and short focused puzzle sessions.

### Is Fibonacci Hex Merge like 2048?

It shares the broad idea of merging numbers, but the rules are different. Fibonacci Hex Merge uses Fibonacci sequence merges on a hex board instead of equal-number merges on a square grid.

### Does it support Game Center?

Yes. Fibonacci Hex Merge supports Game Center leaderboards, so you can compare scores with other players.

### What is the Big Bang ability?

Big Bang is an in-game ability that clears three tiles from the board. It gives you a way to recover space when the board gets tight.
